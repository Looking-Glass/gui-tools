

Prefabs and tools for GUI design for the Looking Glass holographic display.
Requires: Text Mesh Pro and (optionally) the Unity Post Processing package v1


HOW TO USE THE SDF ICON COMPONENT:

Add the SDFIcon.cs component to a UI element with an Image component.


GENERATING SDF TEXTURES:

Open the SDF Generator (Window > SDF Generator) to create an SDF texture, then apply it 
to the SDFIcon component, or use it in a material with the included UI-SDF shader.
